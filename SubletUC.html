SubletUC

Overview

SubletUC is a web application that helps connect students in the surrounding UC area that are looking to sublease their rooms/apartments to other students for short periods of time, focused on the yearly semester schedule. The website acts as a student-only marketplace for them to post and browse through listings.

Solution

Technology Used

  • Blazor WebAssembly & Razor Pages – for building both the frontend and backend using a unified .NET framework.
  • .NET 8 (MVC Architecture) – for structured, scalable application development using the Model-View-Controller pattern.
  • Entity Framework Core – for database access, data manipulation, and schema management through object-relational mapping.
  • Figma – for designing UI/UX wireframes, prototypes, and visual design systems.
  • Fluent UI Blazor – Microsoft’s component library for building accessible and consistent user interface elements.
  • Radzen UI for Blazor – used to extend UI capabilities with advanced components such as forms and dropdowns.
  • Microsoft Azure – for hosting database services, communication services, and security resources.
  • Hybrid Cloud Infrastructure – combines local application servers with cloud-based services for scalability and security.
  • Cloudflare – for domain hosting, DNS management, zero-trust tunnels, and additional security protection.
